# 汽车数据分析系统【sn0505】 python139

## 项目概述

本项目是基于Python开发的汽车数据分析系统，通过集成的多种算法和可视化工具，能够对汽车市场销售数据提供深入分析，帮助用户把握市场动态，预测销量趋势。

## 技术栈

- Python
- Flask
- Bootstrap
- Echarts
- requests
- 预测算法：
 - ARIMA差分自回归移动平均算法
 - 决策树回归算法
 - Ridge岭回归算法

## 运行环境

- Python 3.7
- Flask
- Echarts

## 功能模块

### 汽车总体销量分析

- 展示月、年销量走势的折线图和柱状图，实时把握市场动态。

### 品牌汽车销量对比分析

- 按年份对比各品牌销量（TOP50），以柱状图形式展现。
- 展示热销品牌TOP10的销量占比，以饼图形式呈现。

### 新能源汽车销量预测

- 使用多种算法预测新能源汽车品牌未来月销量，并展示预测结果。

### 交互可视化分析平台

- 使用Flask等工具构建的可视化平台，便于分析不同品牌销量趋势及对比。

## 部署步骤

1. 确保Python环境为3.7版本。
2. 安装必要的依赖库：Flask、Echarts。
3. 部署Web服务，运行Flask应用。
